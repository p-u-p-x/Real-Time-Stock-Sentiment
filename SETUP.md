# Setup Guide

## Prerequisites
- Python 3.8+
- Binance API account
- Reddit API account

## Installation
1. Clone repository
2. Install requirements: `pip install -r requirements.txt`
3. Set up API keys in `.env` file
4. Run data collection: `python main.py`
5. Launch dashboard: `streamlit run streamlit_app.py`

## API Configuration
- Binance: Enable "Reading" permissions only
- Reddit: Create "script" type application
